# UXR AI Articles

A curated site of AI articles from across the web, organized by themes relevant to UX Researchers.

## What this is

A lightweight static website that collects useful AI articles and organizes them under UXR-relevant themes so the team can stay current without the noise.

## Themes

- Trust & Safety
- Human–AI Workflows
- Future of Work
- AI Education
- Evaluating AI Outputs
- AI Technology

👉 https://shashir29.github.io/ai-uxr-hub/

## Suggest an article

Have an article that belongs here? Reach out to me on Teams with:
- Article title
- URL
- Which theme it fits (or suggest a new one)
- A one-line summary of why it's relevant for UXRs

I'll add it to the site within a few days.

## Built with

- HTML / CSS / JS (static site, no backend)
- Python scripts for content ingestion
- GitHub Copilot in VS Code (end-to-end)
